# Nitesh Kumar Verma Portfolio

Personal portfolio website built with Astro and deployed on Cloudflare Pages.

## 🚀 Tech Stack

- **Framework**: Astro 5
- **Deployment**: Cloudflare Pages
- **Styling**: Vanilla CSS + Bootstrap 5
- **Scripts**: Vanilla JavaScript

## 🛠️ Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview build locally
npm run preview

# Deploy to Cloudflare Pages
npm run deploy
```

## 📁 Project Structure

```
src/
├── components/     # Astro components
├── layouts/       # Page layouts
├── pages/         # Route pages
├── scripts/       # JavaScript files
└── styles/        # CSS files
public/
├── images/        # Static images
├── styles/        # Compiled CSS
└── videos/        # Video assets
```

## 🌐 Deployment

The site deploys automatically to Cloudflare Pages on push to main branch.
